# Abstract

## Style Anchor

Write only in polished academic English. Use the supplied Politecnico di Torino theses only as style and tone models: formal but readable prose, clear problem framing, method-first technical specificity, cautious claims, and direct links between data, model, evaluation, and contribution. Do not reuse their wording, research claims, citations, numbers, or chapter content unless the current project sources independently support them.

Prefer "this thesis" for the work as a whole and "the proposed approach/model" for technical contributions. Use "we" sparingly and consistently if the surrounding thesis already uses it. Avoid marketing language; use "novel", "innovative", or "state-of-the-art" only when the thesis substantiates the claim.

## Abstract Shape

Draft one self-contained paragraph of about 250-350 words unless the user or template requires another length. Do not include citations, equations, figure references, footnotes, or bullet points.

Follow this sequence:

1. Open with the research context in one sentence.
2. State the problem or limitation in existing artist similarity analysis.
3. Present the thesis objective.
4. Summarize the proposed method, naming the relevant modalities, embeddings, fusion strategy, model family, training objective, and similarity metric only when they are actually used.
5. State the dataset and evaluation setup at a high level, including sample size only if confirmed by project data.
6. Report the main results in guarded language, using exact metrics only if available.
7. Close with the contribution or implication for artist similarity, MIR, recommendation, or multimodal representation.

## Voice and Tense

Use present tense for what the thesis proposes and demonstrates. Use past tense for completed experiments and data processing. Keep sentences moderately long but controlled; each sentence should add one technical or argumentative step.

Prefer formulations such as:

- "This thesis proposes..."
- "The proposed approach combines..."
- "Experiments conducted on..."
- "The results indicate/suggest/show..."
- "These findings highlight..."

Avoid:

- Unsupported superlatives.
- First-person biographical explanations.
- Implementation details that belong in Methodology.
- Vague result claims such as "good performance" without metrics or comparison.

## Revision Checklist

Ensure the abstract can stand alone for an examiner who has not read the thesis. Check that the central research question, method, data, evaluation metrics, main findings, and contribution are all present. Remove any sentence that only describes the chapter structure.
